# 📝 Social Registration Form

## 📖 Overview
This project is a responsive Social Registration Form built using HTML and CSS. It is designed to simulate a modern sign-up interface commonly used in social media platforms.

The focus of this project is clean UI design, form layout structuring, and responsive styling.

---

## 🎯 Features
- 🧾 Clean and modern registration form layout  
- 📱 Responsive design for mobile and desktop  
- 🎨 Styled input fields and buttons for better UX  
- 🔒 Form structure including name, email, and password fields  
- 🧑‍💻 Simple and user-friendly interface  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3
